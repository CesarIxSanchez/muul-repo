# Project Bootstrap Checklist

Checklist inicial para empezar desarrollo siguiendo lineamientos del proyecto.

## Gobierno tecnico

- Definir convenciones de ramas y PR.
- Definir convencion de commits.
- Definir politica de revision de codigo.

## Backend

- Definir stack final (Node, Python, etc.) y framework.
- Definir contrato de autenticacion (login, registro, refresh, logout).
- Definir contrato de catalogo (lugares y negocios).
- Definir contrato de rutas y logros.
- Definir estrategia de persistencia (DB, migraciones, backups).

## Clientes

- Definir mapa de rutas base para Android, iOS y Web.
- Definir tema visual compartido.
- Definir estrategia de internacionalizacion.
- Definir manejo de estado y capa de datos.

## Calidad

- Configurar CI minima en GitHub Actions.
- Agregar pruebas de smoke por app.
- Agregar pruebas de API en backend.
- Configurar analisis estatico obligatorio en PR.
